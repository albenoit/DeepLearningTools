deeplearningtools.tools.experiment_settings.loadExperimentsSettings(filename, call_from_session_folder=False, restart_from_sessionFolder=None, isServingModel=False)[source]#

Load experiments parameters from the mysettingsxxx.py script. Also mask GPUs to only use the ones specified in the settings file.

Parameters:

  • filename (str) – The settings file. If restarting an interrupted training session, you should target the experiments_settings.py copy available in the experiment folder to restart.

  • call_from_session_folder (bool, optional) – Flag indicating whether the function is called from the session folder. Defaults to False.

  • restart_from_sessionFolder (str, optional) – [OPTIONAL] Set the session folder of a previously interrupted training session to restart.

  • isServingModel (bool, optional) – [OPTIONAL] Set True in the case of using model serving (server or client mode) so that some settings are not checked.

Returns:

The loaded ExperimentSettings object and the session folder.

Return type:

Tuple[ExperimentSettings, str]
